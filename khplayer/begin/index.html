K.H.PLAYER
I. HDSD:
• Nguồn poster chính: ảnh poster chính, nếu không có sẽ dùng ảnh mặc định.
• Tiêu đề: điền tiêu đề video.
• Nguồn poster: nếu không có sẽ dùng ảnh poster chính.
• Nguồn video: direct url của video (xem lưu ý ở dưới)
• Chất lượng, định dạng: nếu không chọn, mặc định 1080p và MP4.
• Tên phụ đề: ngôn ngữ của phụ đề.
• Viết tắt: VN, EN, CN, RU,...
• Nguồn phụ đề: direct url của phụ đề không bị CORS chặn (xem thêm)

II. 4 nút generate code:
• Dữ liệu: Chỉ xuất dữ liệu từ input form phía trên, dùng trong trường hợp muốn thêm video vào K.H.Player playlist có sẵn.
• Full Code: generate dữ liệu và url của K.H.Player/jQuery kèm trong cặp thẻ <script></script>
• ISOLATED: Tính năng này sắp tới sẽ có. Các chế độ generate code kia đều phải load 4 file rời (2 file css, 2 file js của Plyr và K.H.Player phần playlist), điều này làm tăng thời gian load khung player từ 1 đến 2 giây nhưng trông gọn hơn. Code "Cách ly" sẽ generate ra đoạn code hoàn toàn độc lập, kể cả mình xóa 4 file kia đi thì code vẫn dùng bình thường.

III. Lưu ý:
• Cách tạo link direct: Đối với link OneDrive, thêm &download=1 vào cuối link, còn Google Drive nếu file nhẹ hơn 25MB, lấy id của file trong link và đặt vào sau https://drive.google.com/uc?export=download&id=, hơn 25MB thì đặt sau https://direct.gdrive.vip/.
